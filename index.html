MEMO-01. OVERVIEW

물류 현장의 발목을 잡는 '병목 현상' 시각화

수많은 터미널 간 지연 상태 로그를 텍스트로만 쳐다보고 조치하는 비효율을 고쳐보기 위해 작성한 인터랙티브 분석 시트 형태의 웹 애플리케이션이다.

❌ 발견했던 문제상황

매일 수만 건의 배송 지연 로그가 누적되지만, 데이터베이스 구조가 복잡해 실제 물류 정체가 발생한 '진짜 포인트'를 한눈에 발견할 수 없었음.

💡 나의 해결 과정

복잡한 외부 프레임워크 대신 바닐라 JavaScript의 데이터 매핑 루틴을 적용해 로그 데이터를 집계했고, 브라우저 화면상의 격자 맵에 병목 점수에 따른 민트-핑크 경고 컬러 레이어를 씌우는 히트맵 모듈을 개발함.

📊 REAL OUTPUT REPORT
  • 순수 바닐라 모듈 패턴으로 개발하여 무거운 종속 라이브러리 없이 초기 속도를 45% 단축함.
  • 지연 분석 필터 동작을 자동 연산화시켜서 현업 담당자가 병목 지점을 인지하는 시간을 실시간 수준으로 변혁함.
